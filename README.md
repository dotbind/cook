# 開発背景
鈴木大志のWEBエンジニアリングの勉強のため。
4年のギャップを埋めるため、最近のスタンダードなフレームワーク、インフラでWEBアプリをつくってみる

# つくるもの
鈴木大志が作る料理をアップするサイト
日付別で朝昼夜の料理を写真とコメントで投稿

# アーキテクチャ
- Laravel7
- PHP8(詳細バージョン追記予定)
- MySQL8(詳細バージョン追記予定)
- docker-compose(詳細バージョン追記予定)
